Política de Privacidade

1. Introdução

Bem-vindo ao Primeiros Joguinhos (primeirosjoguinhos.pt). Esta política explica de forma simples e transparente como tratamos os dados quando utilizas o nosso site. O site é dirigido a crianças dos 1 aos 4 anos, pelo que o nosso princípio é claro: recolher o mínimo possível de dados.

2. Responsável pelo Tratamento

Este site é um projeto pessoal e sem fins comerciais. Para qualquer questão relacionada com privacidade, podes contactar-nos em: info@primeirosjoguinhos.pt

3. Dados que NÃO recolhemos

Para sermos claros, não recolhemos nem armazenamos nenhum dos seguintes dados:

  • Nome, morada, email ou telemóvel
  • Data de nascimento ou idade da criança
  • Localização geográfica
  • Endereço IP (não é guardado em nenhum sistema nosso)
  • Fotografias, vídeos ou áudio
  • Histórico de navegação ou comportamento

4. Dados técnicos locais (no próprio dispositivo)

Para que alguns jogos funcionem corretamente, podem ser guardados localmente no dispositivo (no browser) alguns dados técnicos mínimos, como:

  • Preferências de jogo (ex.: se já aceitaste o aviso de cookies)
  • Progresso em curso dentro de um jogo

Estes dados nunca saem do teu dispositivo e podem ser apagados a qualquer momento através do browser.

5. Não há publicidade nem rastreadores

Este site não tem publicidade, nem Google Analytics, nem píxeis de redes sociais, nem qualquer rastreador de terceiros. Também não partilhamos dados com ninguém, simplesmente porque não os temos.

6. Alojamento

O site é alojado na plataforma Cloudflare Pages. A Cloudflare pode processar tecnicamente os pedidos feitos ao site (por exemplo, para prevenção de abusos), conforme a sua própria política. Consulta: cloudflare.com/privacypolicy

7. Direitos ao abrigo do RGPD

Como não armazenamos dados pessoais, não há dados para aceder, corrigir ou apagar do nosso lado. Se tiveres dúvidas, contacta-nos no email acima.

8. Crianças

Este site é pensado para crianças pequenas. Ainda assim, recomendamos que os pais/encarregados de educação acompanhem a utilização e configurem as definições adequadas no dispositivo.

9. Alterações

Esta política pode ser atualizada para refletir melhorias ou alterações legais. A versão mais recente estará sempre disponível nesta página.
